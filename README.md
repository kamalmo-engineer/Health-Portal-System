🏥 Secure Health Portal System (C++)
A robust management system for healthcare providers and patients, built with a strong focus on Data Privacy and Object-Oriented Programming (OOP).

🛡️ Core Features
Encrypted Medical Records: Implements a matrix-based encryption/decryption layer to secure sensitive patient diagnoses before storage.

Security-First Access: Features a secure Doctor login with a "3-attempt lockout" mechanism to prevent unauthorized system access.

Data Persistence: Utilizes C++ fstream for reliable file-based database management, ensuring user records are preserved.

OOP Architecture: Clean code structure using Multi-level Inheritance and Encapsulation to manage different user roles (Doctor/Patient).

🛠️ Technical Implementation
Languages: C++

Development Environment: Linux (Ubuntu)

Libraries: <vector>, <fstream>, <string>, <iostream>

Algorithm: Custom Hill Cipher-inspired encryption for medical data.
